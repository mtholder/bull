This is bull, a simulator of protein-coding sequence data under the 
Halpern-Bruno model.

Written by Mark T. Holder

NO WARRANTY (see the text file "COPYING for information on licences and
further info on the LACK of any warranty)


See INSTALL for instructions on building.

See:
    test/cytb-sim.nex for an example of a small input file
    
    test/fullcytb-sim.nex has the parameters for the full cyt-b simulations 
used in the Holder, Zwickl, Dessimoz (2008) paper.
